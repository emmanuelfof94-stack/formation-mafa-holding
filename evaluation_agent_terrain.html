📋 Corrigé — Cas Pratique

A — Profil Marchand

Profil : Essentiel. M. Traoré est un petit commerçant (épicerie de quartier) avec un chiffre d'affaires mensuel estimé à 80 000 XOF, soit un CA annuel d'environ 960 000 XOF. Ce profil correspond à un marchand de proximité dont le besoin principal est de recevoir des paiements clients simplement et gratuitement.

B — Méthode CRAC sur l'Objection "Frais cachés"

C — Creuser : "Quels types de frais vous préoccupent exactement : les frais d'ouverture de compte, les frais sur chaque encaissement, ou autre chose ?" — Identifier précisément la crainte.
R — Reformuler : "Si je comprends bien, votre inquiétude principale est de ne pas payer des frais sur chaque vente que vous réalisez. C'est bien ça ?"
A — Argumenter : "OM Business est totalement GRATUIT à l'ouverture. Les encaissements clients sont à 0 % de frais jusqu'à 50 000 XOF par jour. Au-delà, un taux de seulement 1 % s'applique. Il n'y a aucun frais mensuel fixe."
C — Conclure : "Est-ce que ça répond à votre préoccupation ? Puis-je vous montrer en pratique comment ça fonctionne sur mon téléphone ?"

C — Documents KYC à collecter

1. CNI ou passeport valide (pièce d'identité du gérant)
2. Justificatif de l'activité commerciale (registre de commerce, autorisation d'exercer ou patente)
3. Photo de l'espace commercial (prise via l'application au moment de la visite)

D — Finalisation du Point de Paiement

1. Saisir les données KYC dans l'application mobile de l'agent.
2. Effectuer le linkage QR Code pour associer le compte OM Business de M. Traoré à son QR Code personnalisé.
3. Installer le sticker "PAYER ICI" à un endroit visible dans le commerce.
4. Remettre le QR Code imprimé au marchand.
5. Réaliser une transaction test (1 XOF) pour valider le bon fonctionnement.
6. Prendre une photo de confirmation via l'application.
